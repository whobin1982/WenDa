# 任务-接口精确映射表 v1.0

**适用项目：AI 原生本科专业建设与学生能力成长平台**  
**用途：作为自动开发任务边界、接口实现归属、接口验收失败回溯的权威映射。**

---

## 1. 使用原则

1. 本表覆盖接口文档 v0.2 中的 162 条核心 API。
2. 已排除 `API-DEC-*` 决策项和 `API-FT-*` 后续技术细化项。
3. 禁止使用“相关接口”“全部接口”“等”作为自动开发任务边界。
4. 每个核心 API 至少关联一个 DEV 任务。
5. 接口验收失败时，应按本表回溯到具体 DEV 任务、模块、数据表和验收项。
6. 如后续接口新增、删除或变更，必须同步更新本表。

## 4. 后续维护规则

1. 新增接口时，必须新增 MAP 记录。
2. 删除接口时，必须标记废弃原因和替代接口。

4. 一个 DEV 任务如涉及接口实现，必须列出具体 API 编号。
5. 接口变更必须同步接口文档、详细技术方案、验收清单和本映射表。
